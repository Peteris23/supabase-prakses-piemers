# Programmas prasību specifikācija (PPS)

## Projekta mērķis
Šī ir maza tīmekļa lietotne, kas nolasa un parāda ierakstus no Supabase tabulas.  
Projekts paredzēts kā prakses piemērs: GitHub repozitorijs, Supabase datubāze, publiska GitHub Pages lapa un dokumentācija.  
Mērķis ir parādīt pilnu ceļu no datu glabāšanas līdz lietotājam redzamai lapai.

## Galvenie lietotāji
- Prakses dalībnieks (praktikants).  
- Pasniedzējs/mentors, kas vērtē prakses darbu.  

## Ko lietotājs var darīt
- Atvērt GitHub Pages adresi un apskatīt sarakstu ar ierakstiem (vārds + izveides laiks).  
- Šī versija ļauj tikai nolasīt datus. Nākotnē varētu pievienot ierakstu pievienošanu, labošanu, dzēšanu un lietotāju pieteikšanos.
